# Machine-Learnin🚀 Journey Completed: From Classic ML to Deep Learning! 🧠

I am thrilled to announce that I have successfully completed all the tasks for my Data Science and Machine Learning Internship at  Codveda Technologies ! 🎉

This internship has been an incredible learning curve. Moving from level to level, I got hands-on experience in solving real-world data problems. Here are the major milestones I achieved:

🔹 Level 1: Predictive Modeling
Worked on building robust classification models to predict Customer Churn, learning how to handle data preprocessing and evaluate models effectively.

🔹 Level 2 & 3: SVM & Decision Boundaries
Implemented Support Vector Machines (SVM) on 2D datasets. The most fascinating part was visualizing the mathematical magic—seeing firsthand how a Linear Kernel draws straight lines, while an RBF Kernel bends to create flexible, complex decision boundaries!

🔹 Level 3: Deep Learning & Neural Networks
Took my first leap into Deep Learning! I built a Feed-Forward Neural Network using TensorFlow/Keras to classify handwritten digits (MNIST dataset). Designing the architecture (Input -> Hidden ReLU -> Softmax Output), using backpropagation with the Adam optimizer, and plotting the Epoch vs. Accuracy/Loss graphs was an amazing experience. 📈g-Internship-Tasks
